## ¿Qué es 

Git es un sistema de control de versiones que permite guardar y administrar los cambios que hacemos en un proyecto. Nos ayuda a llevar un historial de modificaciones y a trabajar en equipo sin perder informacion, cada cambio queda registrado con fecha y autor

## Por qué necesitamos Git

Sin Git, cuando varias personas trabajan en el mismo proyecto pueden sobrescribir el trabajo de otros o perder cambios importantes, Git permite que cada persona trabaje de manera organizada y luego combine sus cambios sin conflictos

## Los 3 conceptos clave de Git

1. **Repositorio:** Es la carpeta del proyecto donde Git guarda todo el historial de cambios
2. **Commit:** Es un guardado del proyecto en un momento especifico con un mensaje descriptivo
3. **Rama:** Es una linea de desarrollo independiente que permite trabajar sin afectar la version principal

## Los comandos que aprendí

-git init
-git add
-git commit
-git push
-git pull

## Mi reflexión final

Lo mas importante que aprendi de Git es que facilita mucho el trabajo en equipo y evita perder informacion, ahora entiendo como guardar versiones correctamente y como sincronizar mis cambios con GitHub, Git es una herramienta fundamental para cualquier programador
